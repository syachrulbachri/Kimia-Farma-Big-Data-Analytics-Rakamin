# Kimia Farma Business Performance Analysis (2020-2023)

Proyek ini merupakan bagian dari tantangan Big Data Analytics Intern di Kimia Farma. Tujuan proyek ini adalah untuk menganalisis data transaksi perusahaan dan menyajikannya dalam dashboard interaktif.

## 🔗 Tautan Terkait
- **Dashboard Interactive:** [https://lookerstudio.google.com/reporting/72f522a4-5dd9-4b56-b7d4-0543ac0e3704]
- **Dataset:** BigQuery Public Dataset / Provided Company Data

## 📊 Insight Utama
- Analisis pertumbuhan pendapatan dari tahun ke tahun.
- Pemetaan geografis laba bersih per provinsi di Indonesia.
- Identifikasi anomali pada cabang dengan rating toko tinggi namun rating transaksi rendah.

## 🛠️ Teknologi yang Digunakan
- **SQL (Google BigQuery):** Untuk pemrosesan dan transformasi data.
- **Google Looker Studio:** Untuk visualisasi data interaktif.
